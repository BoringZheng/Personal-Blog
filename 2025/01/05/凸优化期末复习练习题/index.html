Problem 3

Let A∈Rm×nA \in \mathbb{R}^{m \times n} A∈Rm×n be given. Show that exactly one system in each of the following pairs has a solution.

(a)

(I)Ax<0,x≥0\text{(I)} \quad A x < 0, \quad x \geq 0 (I)Ax<0,x≥0 (II)ATy≥0,y≥0,y≠0 \text{(II)} \quad A^T y \geq 0, \quad y \geq 0, \quad y \neq 0 (II)ATy≥0,y≥0,y=0

(I)Ax≥0,Ax≠0 \text{(I)} \quad A x \geq 0, \quad A x \neq 0 (I)Ax≥0,Ax=0 (II)ATy=0,y>0 \text{(II)} \quad A^T y = 0, \quad y > 0 (II)ATy=0,y>0
